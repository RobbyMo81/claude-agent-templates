- `.alpackages/` - Symbol packages for dependencies (gitignored)
- `.snapshots/` - Schema snapshots (gitignored)
- `.output/` - Testing output (gitignored)
- `*.app` - Compiled extension app files (gitignored)
- `rad.json` - Rapid Application Development config (gitignored)
- `*.g.xlf` - Translation base files (gitignored)
- `*.flf` - License files (gitignored)
