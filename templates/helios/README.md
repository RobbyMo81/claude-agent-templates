# Helios - Advanced AI System

Helios is a comprehensive AI reasoning and decision-making system with advanced machine learning capabilities.

## Features

- Advanced neural network training
- Cross-model analytics
- Real-time prediction system
- Metacognitive dashboard
- Memory store with persistence
- Docker containerization support

## Setup

1. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

2. Configure environment:
   ```bash
   cp .env.example .env
   # Edit .env with your configuration
   ```

3. Run the application:
   ```bash
   python server.py
   ```

## Architecture

- **Backend**: Python Flask server with advanced ML capabilities
- **Frontend**: React with TypeScript
- **Database**: SQLite with memory optimization
- **ML Framework**: TensorFlow/PyTorch integration

## Components

- `backend/` - Core AI engine and API
- `components/` - React UI components
- `services/` - API and data services
- `models/` - Trained ML models
- `tests/` - Comprehensive test suite

## Configuration

See `.env.example` for all available configuration options.
